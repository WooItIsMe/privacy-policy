云联安全隐私政策

最后更新日期：2025年6月12日

欢迎使用云联安全（以下简称"本应用"）。我们高度重视您的隐私保护，本隐私政策详细说明了我们如何收集、使用、存储和保护您的个人信息。请在使用本应用前仔细阅读本政策。

1. 信息收集范围

1.1 设备权限相关数据

为实现完整功能，本应用会请求以下设备权限：

  • 相册访问权限：用于读取您选择的图片进行上传，若拒绝将无法使用图片上传功能
  • 相机权限：用于拍摄照片，若拒绝将无法使用拍照功能
  • 位置权限：用于在拍摄的照片中添加地理位置水印信息，若拒绝将无法记录照片拍摄位置
  • 相册写入权限：用于将拍摄的照片或签名保存至您的设备相册，若拒绝将无法保存照片

1.2 面部识别数据

在以下场景我们会进行面部信息采集：

  • 首次使用时拍摄面部照片并上传至服务器建立基准档案
  • 考试/培训/签到环节再次拍摄照片进行实时比对
  • 用于身份核验，防止代考、代签到等作弊行为

所有面部数据仅用于身份验证目的。

1.3 其他可能收集的信息

  • 设备信息：包括设备型号、操作系统版本、唯一设备标识符等
  • 日志数据：应用运行时的错误日志和使用数据
  • 网络信息：IP地址、网络连接类型等

2. 信息使用方式

我们仅会在以下合法基础上使用您的个人信息：

  • 履行与您的合约（提供应用核心功能）
  • 遵守法律义务
  • 保护您或他人的重要利益
  • 出于公共利益或官方授权
  • 我们的正当商业利益（在评估后确认不会损害您的权益）

具体用途包括：

  • 提供照片拍摄、上传和位置标注功能
  • 进行面部识别验证确保考试/培训真实性
  • 改进和优化应用性能
  • 防范欺诈和滥用行为
  • 遵守法律法规要求

3. 数据存储与保护

3.1 存储期限

  • 面部数据：因需要保存面部基准信息用于考试/培训/签到环节的面部对比，所以面部录入处的照片信息会长期保存，除非你重新传入新的面部照片或删除账号
  • 位置信息：仅临时用于照片水印，不长期存储
  • 日志数据：保留30-180天用于故障排查

3.2 安全措施

我们采取行业标准的安全防护措施：

  • 数据传输使用SSL/TLS加密
  • 敏感信息（如面部数据）加密存储
  • 严格的内部访问控制制度
  • 定期安全审计和漏洞扫描

4. 数据共享与披露

我们不与任何第三方共享你的面部照片信息，面部照片会保存在我们的数据库直接进行对比

我们承诺不会出售您的个人信息，仅在以下情况可能共享数据：

  • 服务提供商：与我们有保密协议的第三方服务商（如云存储服务）
  • 法律要求：响应司法机关合法要求
  • 企业交易：如发生合并、收购等情形
  • 保护权益：为防止严重人身伤害或重大欺诈行为

5. 您的权利

根据适用法律，您享有以下权利：

  • 访问权：请求获取我们持有的您的个人信息副本
  • 更正权：要求更正不准确或不完整的个人信息
  • 删除权：在特定情况下要求删除您的数据
  • 限制处理：在特定情况下限制我们处理您的数据
  • 撤回同意：随时在设备设置中关闭各项权限

6. 未成年人保护

本应用不面向16周岁以下用户。如果我们发现无意中收集了未成年人的个人信息，将立即采取措施删除相关数据。

7. 政策更新

我们可能不定期更新本政策。更新后，我们将通过以下一种或多种方式通知您：

  • 在应用内显著位置发布通知
  • 更新政策顶部的"最后更新日期"
  • 在适用情况下通过电子邮件通知
